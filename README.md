# CodexCraft

## Build Prerequisites
- **Windows SDK**: Install the latest Windows 10 or 11 SDK to provide the required headers, libraries, and build tools.
- **DirectX Runtime**: Install the DirectX End-User Runtimes (June 2010) or the latest DirectX runtime available to ensure all DirectX components are present.

## Getting Started
1. Clone or download this repository to your local machine.
2. Ensure the prerequisites above are installed on your development machine.
3. Open `CodexCraft.sln` in Visual Studio by double-clicking the file or selecting **File > Open > Project/Solution...** from within Visual Studio and browsing to the solution file.
4. Set the desired configuration (e.g., `Debug` or `Release`) and target platform, then build and run the project.
